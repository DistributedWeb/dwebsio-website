Getting Started Guide

Get started with dWeb in minutes!

Below is a simple 3-step instruction on how to get started using the dWeb.
  • 1. Download dBrowser

    Download the official web browser for Web 3.0 and the dWeb protocol. It's free and can be downloaded from here.

  • 2. Install dDNS

    Install dDNS by either downloading dDNS Desktop or installing dDNS manually. Instructions for installing dDNS can be found here or you can download dDNS Desktop here


3. Start Using dWeb!

Perform the following tests:

1. Open dBrowser, and copy and paste the following link into the address bar: dweb://dwebs.io
2. Within dBrowser, go to the following address:
http://go.dweb


If both of these links resolve, you're set and ready to go!
