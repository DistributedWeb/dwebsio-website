dDatabase

dDatabase's core components and modules that bring dWeb's distributed database framework for distributed applications to life.
dWeb Core Components

dDatabase's Distributed Database Components & Libraries

Blockchains are not the only way to create dApps, when distributed and decentralized data is the use case. dDatabase's framework brings to life 3 different types of distributed databases, to power dApps like dStatus and dPress.

dDatabase Core

Distributed and immutable database framework for the distributed web.

dDatabase Core Documentation

dSiteDB

A distributed database that reads and writes records on dweb:// websites..

dSiteDB Documentation

dAppDB

DappDB is a scalable peer-to-peer key-value database for the distributed web protocol (dweb://) and other distributed web protocols.

dAppDB Documentation

dDatabase Protocol Buffers

dDatabase's protocol stream library.

dDatabase Protocol Buffers Documentation

dDatabase Cryptography

dDatabase cryptography library for dDatabase keys.

dDatabase Cryptography Documentation

dDatabase Daemon

dDatabase daemon for sharing multiple dDrives and dDatabases at once.

dDatabase Daemon Documentation

dDatabase Vaultr

Library For Creating and Managing dDatabase and dDrive Vaults.

dDatabase Vaultr Documentation

dDatabase Tree Storage

A module that stores dDatabase metadata trees using a storage provider.

dDatabase Tree Storage Documentation
