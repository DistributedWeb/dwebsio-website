FlockCore Peering Libraries

dWeb's foundational libraries for peering are referred to as FlockCore. A group of peers aka the "Flock", gather around dSites and dApps in an effort to seed a distributed website or distribute dapp to more users, faster and more efficiently. Think of it like a website or an application running on more than one server. When it comes to Web 3.0, a peer is a server, that you never pay for.

FlockCore Core

The core library for FlockCore, used as the defacto method within dWeb-ready dSites and dApps, like dBrowser, dStatus or dPress to join a peer-to-peer flock for dDatabase or dDrive feeds.

FlockCore Core Documentation

FlockCore Channels

Search for a key across multiple dWeb revelation networks and find peers who answer. Currently searches across and advertises on the Bittorrent DHT, centralized dWeb DNS servers and Multicast DNS simultaneously.

FlockCore Channels Documentation

FlockCore Network Presets

A library containing live dWeb revelation tower locations along with BitTorrent DHT towers, so dWeb-ready dApps like dBrowser, can identify a dWeb key and find the location of a dSite and/or dApp it represents.

FlockCore Network Presets Documentation

FlockCore Revelation

FlockCore's library used to find and connect to other peers on the dWeb. Utilizes FlockCore's channel library simultaneously.

FlockCore Revelation Documentation

FlockCore Server

A tcp/udp server used for announcing a dSite or dApp to the dWeb netowrk. Uses FlockCore Channel library for announcements.

FlockCore Revelation Documentation
